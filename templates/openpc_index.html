同志社パソコン利用状況

概要 全件取得
API_URL http://hack.doshisha.work/openpc/api/v1/'campusID'/
campusID 今出川 => 1, 京田辺 => 2
説明 キャンパスごとの空き状況を取得
サンプル 京田辺キャンパスの利用状況
形式 "data": { "campus": "キャンパス名", "date": "取得したときの時刻" }, "status": { "教室名": "閉室" or "教室名": { "free": "空きPC数", "max": "全PC台数" } }
形式に関しては後で綺麗に書き直す
概要 開き教室検索
API_URL http://hack.doshisha.work/openpc/api/v1/'campusID'/open
説明 キャンパスごとに開いている教室の状況ののみを取得
サンプル 今出川キャンパスの開き教室状況
形式 "data": { "campus": "キャンパス名", "date": "取得したときの時刻" }, "status": { "教室名": { "free": "空きPC数", "max": "全PC台数" } }
