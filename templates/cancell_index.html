同志社休講情報

概要 全件取得
API_URL http://hack.doshisha.work/cancell/api/v1/'campusID'/'target_day'
campusID 今出川 => 1, 京田辺 => 2, 大学院 => 3
target_day 今日 => today, 明日 => tomorrow
説明 指定日におけるキャンパスごとの休講情報を取得
サンプル 京田辺キャンパスの今日の休講情報
形式 "data": { "campus": "キャンパス名", "date": "取得したときの時刻" }, "cancelled_classes": { "Index": { "class_teacher": "担当教授", "class_cause": "休講理由", "class_hour": "時間", "class_name": "授業名" } }
